# GraphQl Boilerplate with Prisma

A Simple Boilerplate with prisma and graphql-yoga.

## Installation

Use the package manager npm to install.

```bash
npm install
```

## To run Prisma server

```bash
cd prisma
docker-compose up -d
prisma deploy
```

## To run the server

```bash 
npm run dev - to run the dev server
npm run test - to run tests
```
